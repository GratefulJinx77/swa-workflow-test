# swa-workflow-test - Gemini Context

## Overview
Azure Static Web App testing environment with comprehensive security assessment tools and M365 integration testing.

## Key Commands
```bash
# Run security validation
~/alpha-root/swa-workflow-test/azure-security-validation.sh

# View security assessment
cat ~/alpha-root/swa-workflow-test/azure-security-assessment.md

# Check M365 audit report
cat ~/alpha-root/swa-workflow-test/M365-Jinks-Labs-Audit-Report.md

# View security dashboard
# (Open in browser: security-dashboard.html)

# Check remediation status
cat ~/alpha-root/swa-workflow-test/remediation-tracker.md

# Quick remediation guide
cat ~/alpha-root/swa-workflow-test/QUICKSTART-SECURITY-REMEDIATION.md
```

## Security Tools
- **azure-security-validation.sh**: Automated security scanning
- **security-dashboard.html**: Visual reporting interface
- **Audit reports**: M365 and Azure security findings
- **Remediation tracking**: Issue resolution workflow

## Integration Points
- Azure Static Web Apps deployment testing
- Microsoft 365 integration validation
- Security baseline validation for Jinks Labs infrastructure
